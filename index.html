Secure Offline OpenPGP Toolkit

A set of offline OpenPGP tools for encryption, decryption, signing, verification, and key generation.
All operations run entirely in your browser — no uploads, no logging, no tracking.

Security notes

  • No server communication or upload of any kind.
  • No analytics, tracking scripts, or external requests.
  • All cryptographic operations use OpenPGP.js locally.
  • You can save this project and run it fully offline.
