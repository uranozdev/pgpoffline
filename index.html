Secure Offline OpenPGP Toolkit

A set of offline OpenPGP tools for encryption, decryption, signing, verification, and key generation.
All operations run entirely inside your browser. No data is transmitted or stored anywhere.

Security notes

  • No server communication or upload of any kind.
  • No analytics, tracking scripts, or network requests.
  • All data is processed locally using OpenPGP.js.
  • You can save this folder and run everything fully offline.
