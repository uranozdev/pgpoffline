# Contributing to PGP Offline Tools

## Guidelines

- All comments, documentation, and commits must be in English.  
- Keep code simple and easy to audit.  
- Avoid adding dependencies unless necessary.  

## Development

There is no build step.  
Open `index.html` directly or serve the directory with a simple HTTP server.

## Pull Requests

1. Fork the repository.  
2. Create a feature branch.  
3. Submit a PR with a clear description and rationale.
